# LINE登録特典 特設サイト 設計ドキュメント

作成日：2026-05-21  
プロジェクト：AIリーダーキャンプ（ALC）/ プロジェクトF

---

## 概要

LINE登録後に届くURLとして、2大特典をまとめた特設サイトを作る。
ターゲットはAI初心者の女性起業家（30〜50代）。プロンプト集をその日から使ってもらい、「AIって便利！」という体験を作ることが目的。

---

## ゴール

LINE登録した人が以下の状態になること：

1. 「これだけでも価値がある」と感じる
2. ChatGPTをすぐ使ってみたくなる
3. 「もっと学びたい」気持ちが生まれる
4. LINEをまた見てくれるようになる（コース案内を受け取る準備）

---

## 動線設計

```
広告（Instagram等）
  ↓
AI LABO LP（内容・信頼・コース紹介）
  ↓
LINE登録
  ↓
特設サイト URL が届く（配布方法は未定：配信 or リッチメニュー）
  ↓
特設サイト（2大特典を受け取る・体験する）
  ↓
「LINEをチェックしてね」← ここからコース案内が届く
  ↓
LINE（コース案内） ← 未決定
  ↓
各コース申し込み ← 未決定
  ↓
ALC オファー（コース参加者へ・将来）← 特設サイトには出さない
```

## 特設サイト構成（present.html）

### ページ構成

```
1. ヒーロー
   - タイトル：「2大特典をお受け取りください」
   - サブ：「LINE登録ありがとうございます」
   - カラー：ミント(#5abfbf) × ゴールド(#b8965a)

2. 特典① プロンプト集30選
   - 説明文
   - 6カテゴリ × 5プロンプト = 30個
   - 各プロンプト：タイトル / 活用シーン / 本文 / コピーボタン / ポイント

3. 特典② AIロードマップ動画
   - 動画埋め込み（YouTube / Vimeo）※URLは後で差し込み
   - 動画の見どころ説明

4. フッター CTA
   - 「LINEをチェックしてね」
   - LINEへの誘導リンク（URLは後で差し込み）
```

### プロンプトカテゴリ（6カテゴリ × 5個）

| # | カテゴリ | プロンプト一覧 |
|---|---|---|
| 1 | SNS発信 | Instagramフィード / リール台本 / ストーリーズネタ / キャッチコピー / ハッシュタグ |
| 2 | LINE配信 | LINE登録促進文 / ステップ配信 / 締切案内 / セミナー前リマインド / 個別相談案内 |
| 3 | セミナー・講座 | セミナータイトル / 告知文 / セミナー構成 / オファー導入文 / クロージング文 |
| 4 | 提案書・営業 | 提案書構成 / 見積書説明 / サービス紹介文 / お客様返信文 / ヒアリング質問 |
| 5 | 顧客対応 | FAQ / お礼メール / リマインドメール / キャンセル対応 / アフターフォロー |
| 6 | 商品設計・マーケ | ターゲット設定 / 商品コンセプト / ベネフィット整理 / LP構成 / 特典アイデア |

## 動画台本 構成（roadmap-script.md）

10分・5ステップ・AI初心者向け

```
オープニング（1分）
  - 自己紹介・この動画で学べること
  - 「AIって何から学べばいい？」の答えを出す

STEP1：ChatGPT 基礎（2分）
  - 会話・文章作成・アイデア出し
  - 「まずここから始めよう」

STEP2：実践活用（2分）
  - SNS・提案書・セミナー
  - 「毎日の仕事にすぐ使える」

STEP3：応用（2分）
  - 画像生成・動画生成・デザイン

STEP4：自動化（1.5分）
  - LINE・ステップ配信・業務効率化

STEP5：AIエージェント（1分）
  - Claude Code・自動化システム・Web制作
  - 「ここまで来たらすごい！」

クロージング（0.5分）
  - 「今日から始めよう」
  - LINEをチェックしてね
```

- CTA：LINE登録ボタン
