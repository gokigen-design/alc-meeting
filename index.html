01 議論の出発点（なぜ今これをやるのか）
野田さん（ALC運営者）のスペックが現在の女性起業家層に対して高すぎ、ALCにいきなり入るには距離が遠すぎる、という課題認識からスタート。ALC本体の手前に「階段」となるフロント商品群が必要という結論に至った。

現状の課題

  • 野田さんのスペックが高すぎて、今の女性起業家との距離が遠い
  • いきなりAI（ALC）を学ぶのはハードルが高い
  • 受講希望者が「今すぐ目に見える成果」を欲しがっている
  • 野田さん1人での集客・広告運用には限界がある
  • 外（自分のリーチ外）に出ていかないと、これ以上は広がらない
「すごい何がやばいかっていうと、やっぱ野田さんのスペックが高すぎて、今の女性起業家と野田さんがやっぱ遠すぎるんですよ。だから階段が必要だと思ってて。」

過去の成功体験からの確信

過去にも「業界を変えたい」という想いで、1個の講座に収まらない仕組みを作ったことで、ステークホルダーが生まれ、プラットフォームができ、右腕とホルダーがマッチングできるようになり、大きく成長した実績がある。仕組みを作ったから、そうなった。

「私はその業界を変えたいの。だから1個の講座で収まるとかじゃなくて、ステークホルダーが生まれてプラットフォームができて、右腕さんとホルダーさんがマッチングできるようになって、すごい大きくなった。仕組みを作ったから、そうなった。」
02 プロジェクトのコア構想

コンセプト：女性起業家向けフロント講座プラットフォーム

  • ALC本体に行く前段階として、フロント講座を10個ほど用意する
  • サイト上で受講者が自分に必要な講座を選べる
  • 各メンバーがそれぞれの強み（LINE、デザイン、LP、AIなど）を活かして講座化

「今回もやっぱみんなやっぱスキルがあるわけ。あきちゃんだったらLINEとか、ちゃかなさんだったらデザインとか。そういうのをもっと生かしながら、でもみんなでプロジェクトを回したらどうかと思ってて。」

03 収益構造・経済圏設計

フロント講座群を入口に、ALC本体・クライアントワークまで含めた経済圏ごと作るのがゴール。1個で勝負するのではなく、複数のフロントを用意して全員で勝っていく構造にする。

マネタイズの流れ

1外部プロデューサー（カシーさん）が広告を運用し、各フロント講座に集客
2受講者がフロント講座を受講（10本から自分に必要なものを選ぶ）
3フロント修了者をALC本体へアップセル
4ALCはアフィリエート的な仕組みで野田さんが本体を運営
5受講者は独自にクライアントワークを取ってもOK（自由）
6講師として育った人は自分も講師として登壇するルートへ
「うちのプロデューサーのカシーさんが言ってて、今の時代何が当たるかわかんない。だから1個で勝負するよりも何個もあった方がいい。1個当たったら、そっから人が流れるから、みんなで勝っていける。」

02 現時点の懸念点
懸念 1 ｜ Claude Code 未導入者への対応
  • Claude Code の導入が初めてで不安な方がいる
  • 「導入済み」でも実はデスクトップアプリのみ、というケースがある
  • ハンズオン時にサポートが必要になる可能性
懸念 2 ｜ 複数講座に参加する場合の運用
  • 毎回の講座に導入費を含めるのか？
  • 導入済みの方には割引するのか？
  • 全講座に導入サポートを統一含有するのか？
03 Claude Code 導入サポートの方向性（検討中）
A 案
全コースに導入サポートを含める
  • 参加者全員が確実にスタートできる
  • 講座内容に集中しやすい
  • 価格設計が複雑になる可能性
B 案
導入サポートのみの単独コースを作る
  • 既導入者は各講座に直接参加できる
  • 導入コストを1回にまとめられる
  • 未導入者の参加ハードルが上がるリスク
04 特別企画案
注目
野田さんによる「LINE ハーネス 導入スペシャル会」
野田さん主導でLINEハーネスの導入会・Q&A・活用紹介を開催。YouTubeでも学べるが「一緒に設定する・質問できる・強制的に環境を整える」場としての価値が高い。
アイデア（検討中）
「AI の祭典」リアルイベント案
Claude 導入 + 各コース体験 + LINEハーネス導入をセットにした大型リアルイベント案。アイデアとしては面白いが、現時点では難しいかも…というニュアンスで記録。
— Claude 導入を一気に完了できる（つまずきをその場で解決）
— 盛り上がりを"見せる"ことで認知・話題拡散につながる
— イベント自体が集客の入口になる可能性がある
懸念点
— 大人数のイベント形式だとサポートが手薄になりやすく、参加者一人ひとりへの対応が難しくなる可能性がある。
— 難易度の高いコンテンツが組み込めなくなる（イベント形式だと内容が制限される）。
— 「その日・その場で完結」するものに限定されてしまい、深い学びに繋がりにくい。
各メンバーの声
一人での開催ではなくコラボで開催したいという意見が複数あった。特にClaude導入部分への不安が大きく、サポート体制を複数人で組みたいというニーズがある。
